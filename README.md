# LLM Engineering (Udemy coursework)

This repository contains my hands-on implementations, challenge solutions, and projects from Ed Donner's *AI Engineer Core Track: LLM Engineering, RAG, QLoRA, Agents Course*.

## Key Skills & Concepts Mastered
- **Agentic Architecture:** Built autonomous LLM agents utilizing tool-calling loops (Reasoning + Acting / ReAct framework).
- **Tool & API Integration:** Created custom tools allowing models to interface with external databases and web APIs.
- **Context & Memory Management:** Implemented short-term and persistent memory patterns for multi-turn conversations.
- **Model Optimization:** Experimented with system prompting, structured outputs, and performance tuning (temperature, system constraints).

## Tech Stack
- **Language:** Python
- **Frameworks/APIs:** OpenAI API, Anthropic Claude, `python-dotenv`
- **Concepts:** Model Context Protocol (MCP), ReAct Loop, Function Calling
